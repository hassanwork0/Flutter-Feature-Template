# Flutter Feature Template
My Template for Flutter Features. not included with core or code, its just comments

```
└── 📁Feature_name
    └── 📁data
        └── 📁implements
            └── implements.dart
        └── 📁models
            └── models.dart
        └── 📁sources
            └── sources.dart
    └── 📁domain
        └── 📁entities
            └── entities.dart
        └── 📁repositories
            └── repositories.dart
        └── 📁usecases
            └── add_comment.dart
    └── 📁presentation
        └── 📁Bloc
            └── cubit.dart
            └── Feature_bloc.dart
            └── Feature_event.dart
            └── Feature_state.dart
        └── 📁screens
            └── screens here.txt
        └── 📁widgets
            └── widgets here.txt
```
